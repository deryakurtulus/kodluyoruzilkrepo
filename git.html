GIT KOMUTLARI

  • git init

    • Açıklama: Yeni repo açmak için kullanılır.
    Kullanımı: git init repository ad

  • git add

    • Açıklama: Repoya bir dosya eklemek için kullanılır.
    Kullanımı: git add dosya-adi

  • git status

    • Açıklama:İşlenmesi gereken tüm dosyaları listeler.
    Kullanımı: git status

  • git commit

    • Açıklama: Dosyayı sürüm geçmişinde kalıcı olarak kaydeder veya anlık görüntüler.
    Kullanımı: git commit -m “Commit mesajınız”

  • git branch

    • Açıklama: Geçerli depodaki tüm yerel dalları listeler.
    Kullanımı: git branch
    yeni bir dal oluşturmak için: git branch [branch name]

  • git diff

    • Açıklama: Henüz aşamalı olmayan dosya farklılıklarını gösterir.
    Kullanımı: git diff

  • git config

    • Açıklama: v-Verilen değerler ile birlikte kullanılacak yazar adını ve e-posta adresini sırasıyla ayarlar.
    Kullanımı: git config –global user.name “isim” git config –global user.email “e-posta adresi]”

  • git clone

    • Açıklama: Mevcut bir URL'den bir repo elde etmek için kullanılır.
    Kullanımı: git clone url

  • git reset

    • Açıklama: Dosyanın aşamasını kaldırır, ancak dosya içeriğini korur.
    Kullanımı: git reset dosya-adi

  • git log

    Açıklama: Geçerli branchın sürüm geçmişini listelemek için kullanılır.
    Kullanımı: git log

  • git remote

    • Açıklama: Local repoyu remote repoya bağlar.
    Kullanımı: git remote add [variable name] [Uzak Sunucu Linki]

  • git push

    • Açıklama: Ana branchdan taahhüt edilen değişikliklerini remote repoya gönderir. Kullanımı: git push [variable name] master

  • git checkout

    • Açıklama: Branchlar arası geçiş için kullanılır.
    Kullanımı: git checkout [branch name]

  • git pull

    • Açıklama: remote repodaki bilgileri local repoya getirir.
    Kullanımı: git pull repo linki
